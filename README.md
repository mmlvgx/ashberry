# ashberry
A tool for searching websites by generating random URLs
# Requirements
`git` `make` `python3` `python3-pip`
## Ubuntu
`sudo apt install git`\
`sudo apt install python3-pip`
# Using
## Linux
`git clone https://github.com/mmlvgx/ashberry.git`\
`cd ashberry`\
`make setup`\
`make run`
